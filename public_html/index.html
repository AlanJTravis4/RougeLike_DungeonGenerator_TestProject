WillowTree Rougelike Dungeon Generator

The following example generates a random dungeon similar to those found in Rougelike games.

The dungeon generated is very much simplified from what you would find with a full Rougelike game. The intent
however is a proof of concept, and scalable foundation. As such the main dungeon environmental tile types are limited to the following:
  • rock "#"
  • dirt floor "."
There are some additional items and enemies that can be encountered within the dungeon.
These include the following:
  • Items:
    • stairs/ladder up ">"
    • stairs/ladder down "<"
    • treasure chests "$"
    • Traps:
      • FirePits "f"
      • BearPits "b"
      • BeerPits "r"
      • Snare "s"
  • Enemies:
    • lizard "~"
    • troll "%"
    • dragon "D"
What's presented is a SVG image representing a 2D array of tiles. SVG was chosen for this example because it allows:
- Clean presentation of the dungeon
- A foundation which could also support User/AI interaction to play the game

By default a dungeon with width/height dimensions of 800x600 and tiles sized at 15x15 will be generated.
*Note* for cases where the dungeon dimension is not evenly divisable by the tile dimension, the number of
possible tiles for that dimension is calculated by simple integer division which truncates any decimals.
ex) 800 / 15 == 53 possible tiles wide for the dungeon.

If desired you can change the default width/height of the generated dungeon as well as the size of individual tiles.
Once satisfied with the input parameters, click the the button "Generate Random Dungeon".
